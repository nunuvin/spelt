# spelt

A Wordle clone built and deployed on GitHub Pages.

## Play Online

[https://nunuvin.github.io/spelt](https://nunuvin.github.io/spelt)

## About

Spelt is a web-based recreation of the popular word-guessing game Wordle. Guess the word in six attempts and test your vocabulary skills.

## Features

- Daily word challenges
- Responsive design
- Works on desktop and mobile browsers

## Getting Started

To run locally, clone the repository and open `index.html` in your browser.

```bash
git clone https://github.com/nunuvin/spelt.git
cd spelt
```
